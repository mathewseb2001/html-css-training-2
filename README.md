# GG Clone Website
A clone of the GG Hospital Website made as part of the HTML-CSS Training Section for ILP.
## Contents Included
The page has the following contents included:
1. Navigation Bar
2. Book Appointment
3. Doctors Listed
4. Departments Listed
5. Why GG is better section
6. Facilities and Activities
7. You May Be Interested In section
8. Footer
9. Newsletter
